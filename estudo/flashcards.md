## Pergunta: Em qual laboratório o sistema Unix foi originalmente criado em 1969?
Resposta: Bell Labs (pertencente à AT&T na época).
## Pergunta: Quem foram os dois principais criadores do sistema Unix?
Resposta: Ken Thompson e Dennis Ritchie.
## Pergunta: Qual linguagem de programação foi desenvolvida por Dennis Ritchie para tornar o Unix portátil entre diferentes arquiteturas?
Resposta: Linguagem C.
## Pergunta: O que significa o acrônimo recursivo GNU?
Resposta: GNU is Not Unix (GNU não é Unix).
## Pergunta: Quem fundou o projeto GNU em 1983 e a Free Software Foundation (FSF)?
Resposta: Richard Stallman.
## Pergunta: Qual era o principal objetivo de Richard Stallman ao iniciar o projeto GNU?
Resposta: Criar um sistema operacional compatível com Unix que fosse completamente livre.
## Pergunta: Quem desenvolveu o kernel Linux em 1991 como um estudante na Universidade de Helsinki?
Resposta: Linus Torvalds.
## Pergunta: "Qual sistema operacional educacional, criado por Andrew Tanenbaum, serviu de inspiração inicial para Linus Torvalds?"
Resposta: Minix.
## Pergunta: Como é tecnicamente chamado o sistema operacional completo que combina o kernel de Linus Torvalds com as ferramentas do projeto GNU?
Resposta: GNU/Linux.
## Pergunta: O que define uma 'distribuição' Linux (distro)?
Resposta: "Um conjunto que reúne o kernel Linux, utilitários GNU, scripts de instalação e softwares adicionais."
## Pergunta: Quais são as três distribuições 'mãe' ou base citadas como as mais antigas ainda em atividade?
Resposta: "Slackware, Debian e Red Hat."
## Pergunta: Qual é o shell padrão para usuários no Red Hat Enterprise Linux (RHEL)?
Resposta: Bash (GNU Bourne-Again Shell).
## Pergunta: "No prompt do shell, qual caractere indica que o usuário logado é um usuário comum?"
Resposta: O caractere $.
## Pergunta: "No prompt do shell, qual caractere indica que o usuário possui privilégios de superusuário (root)?"
Resposta: O caractere #.
## Pergunta: Qual combinação de teclas permite acessar o primeiro console virtual (gráfico) no RHEL 7?
Resposta: Ctrl+Alt+F1.
## Pergunta: Quais teclas são usadas para alternar entre os consoles virtuais de modo texto (2 a 6)?
Resposta: Ctrl+Alt+F2 até F6.
## Pergunta: Quais são as três partes básicas de um comando inserido no shell?
Resposta: "Comando, opções e argumentos."
## Pergunta: Qual opção padrão a maioria dos comandos aceita para exibir uma descrição de uso e sintaxe?
Resposta: A opção --help.
## Pergunta: Qual comando é utilizado para encerrar uma sessão de shell atual?
Resposta: O comando exit (ou o atalho Ctrl+D).
## Pergunta: Qual é o ambiente de desktop padrão no Red Hat Enterprise Linux 7?
Resposta: GNOME 3.
## Pergunta: "No GNOME 3, qual tecla (também chamada de tecla Windows) abre o 'Activities Overview'?"
Resposta: A tecla Super.
## Pergunta: Qual parte da interface do GNOME fornece acesso rápido aos diretórios do usuário e compartilhamentos de rede?
Resposta: Menu Places (Locais).
## Pergunta: "No GNOME, qual combinação de teclas abre a bandeja de mensagens para revisar notificações?"
Resposta: Super+M.
## Pergunta: Qual atalho de teclado permite alternar sequencialmente entre áreas de trabalho (workspaces) no GNOME?
Resposta: Ctrl+Alt+Seta para Cima ou Seta para Baixo.
## Pergunta: Qual comando exibe a data e hora atual do sistema?
Resposta: O comando date.
## Pergunta: Qual argumento do comando date formata a saída para mostrar apenas a hora atual?
Resposta: O argumento +%R.
## Pergunta: Qual comando é utilizado para um usuário alterar sua própria senha no Linux?
Resposta: O comando passwd.
## Pergunta: Qual é a função do comando file?
Resposta: "Escaneia o conteúdo de um arquivo para determinar seu tipo (ex: texto ASCII, diretório, executável)."
## Pergunta: "Por padrão, quantas linhas os comandos head e tail exibem de um arquivo?"
Resposta: 10 linhas.
