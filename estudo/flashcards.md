## Pergunta: Em qual laboratório o sistema Unix foi originalmente criado em 1969?
Resposta: Bell Labs (pertencente à AT&T na época).

Quem foram os dois principais criadores do sistema Unix?,Ken Thompson e Dennis Ritchie.
Qual linguagem de programação foi desenvolvida por Dennis Ritchie para tornar o Unix portátil entre diferentes arquiteturas?,Linguagem C.
O que significa o acrônimo recursivo GNU?,GNU is Not Unix (GNU não é Unix).
Quem fundou o projeto GNU em 1983 e a Free Software Foundation (FSF)?,Richard Stallman.
Qual era o principal objetivo de Richard Stallman ao iniciar o projeto GNU?,Criar um sistema operacional compatível com Unix que fosse completamente livre.
Quem desenvolveu o kernel Linux em 1991 como um estudante na Universidade de Helsinki?,Linus Torvalds.
"Qual sistema operacional educacional, criado por Andrew Tanenbaum, serviu de inspiração inicial para Linus Torvalds?",Minix.
Como é tecnicamente chamado o sistema operacional completo que combina o kernel de Linus Torvalds com as ferramentas do projeto GNU?,GNU/Linux.
O que define uma 'distribuição' Linux (distro)?,"Um conjunto que reúne o kernel Linux, utilitários GNU, scripts de instalação e softwares adicionais."
Quais são as três distribuições 'mãe' ou base citadas como as mais antigas ainda em atividade?,"Slackware, Debian e Red Hat."
Qual é o shell padrão para usuários no Red Hat Enterprise Linux (RHEL)?,Bash (GNU Bourne-Again Shell).
"No prompt do shell, qual caractere indica que o usuário logado é um usuário comum?",O caractere $.
"No prompt do shell, qual caractere indica que o usuário possui privilégios de superusuário (root)?",O caractere #.
Qual combinação de teclas permite acessar o primeiro console virtual (gráfico) no RHEL 7?,Ctrl+Alt+F1.
Quais teclas são usadas para alternar entre os consoles virtuais de modo texto (2 a 6)?,Ctrl+Alt+F2 até F6.
Quais são as três partes básicas de um comando inserido no shell?,"Comando, opções e argumentos."
Qual opção padrão a maioria dos comandos aceita para exibir uma descrição de uso e sintaxe?,A opção --help.
Qual comando é utilizado para encerrar uma sessão de shell atual?,O comando exit (ou o atalho Ctrl+D).
Qual é o ambiente de desktop padrão no Red Hat Enterprise Linux 7?,GNOME 3.
"No GNOME 3, qual tecla (também chamada de tecla Windows) abre o 'Activities Overview'?",A tecla Super.
Qual parte da interface do GNOME fornece acesso rápido aos diretórios do usuário e compartilhamentos de rede?,Menu Places (Locais).
"No GNOME, qual combinação de teclas abre a bandeja de mensagens para revisar notificações?",Super+M.
Qual atalho de teclado permite alternar sequencialmente entre áreas de trabalho (workspaces) no GNOME?,Ctrl+Alt+Seta para Cima ou Seta para Baixo.
Qual comando exibe a data e hora atual do sistema?,O comando date.
Qual argumento do comando date formata a saída para mostrar apenas a hora atual?,O argumento +%R.
Qual comando é utilizado para um usuário alterar sua própria senha no Linux?,O comando passwd.
Qual é a função do comando file?,"Escaneia o conteúdo de um arquivo para determinar seu tipo (ex: texto ASCII, diretório, executável)."
"Por padrão, quantas linhas os comandos head e tail exibem de um arquivo?",10 linhas.
Como se especifica a exibição de exatamente 3 linhas ao usar o comando tail?,Utilizando a opção -n 3 (ex: tail -n 3 arquivo).
"Qual comando conta o número de linhas, palavras e caracteres em um arquivo?",O comando wc.
"No comando wc, qual opção exibe apenas o número de linhas de um arquivo?",A opção -l.
O que acontece ao pressionar a tecla Tab uma vez durante a digitação de um comando?,O shell tenta completar o nome do comando ou do arquivo se ele for único.
O que o shell faz se a tecla Tab for pressionada duas vezes quando há múltiplas correspondências?,Exibe uma lista de todos os comandos ou arquivos que começam com os caracteres já digitados.
Qual comando exibe a lista de comandos executados anteriormente pelo usuário?,O comando history.
Qual metactaractere do Bash permite reexecutar um comando específico da lista de histórico pelo seu número?,O ponto de exclamação (ex: !26).
Como reexecutar o comando mais recente que começa com a string 'ls'?,Digitando !ls.
Qual atalho de edição de linha do Bash move o cursor para o início da linha de comando?,Ctrl+a.
Qual atalho de edição de linha do Bash move o cursor para o final da linha de comando?,Ctrl+e.
Qual comando apaga todo o texto do cursor até o início da linha de comando?,Ctrl+u.
Qual comando apaga todo o texto do cursor até o final da linha de comando?,Ctrl+k.
Qual atalho permite realizar uma busca interativa no histórico de comandos por um padrão?,Ctrl+r.
Qual combinação de teclas copia o último argumento do comando anterior para a posição atual do cursor?,Esc+.
Qual caractere atua como separador de comandos quando se deseja executar mais de um comando na mesma linha?,O ponto e vírgula (;).
Qual foi a 'gota d'água' para Richard Stallman decidir criar o sistema GNU?,A recusa de uma empresa em fornecer o código-fonte do driver de uma impressora para correção de erros.
Qual universidade é o berço do BSD (Berkeley Software Distribution)?,Universidade da Califórnia em Berkeley.
Qual é a base da distribuição Ubuntu?,Debian.
Qual bilionário e turista espacial sul-africano fundou a Canonical e criou o Ubuntu?,Mark Shuttleworth.
O que o termo africano 'Ubuntu' significa em essência?,Humanidade para com os outros (ou 'Sou o que sou pelo que todos somos').
Qual distribuição é conhecida por não ter instalador e exigir que o usuário compile tudo manualmente?,Gentoo.
Qual o nome da versão do Linux Mint baseada diretamente no Debian em vez do Ubuntu?,LMDE (Linux Mint Debian Edition).
Em que ano Linus Torvalds lançou a primeira versão do kernel Linux?,1991.
Qual projeto de kernel do GNU ainda está em desenvolvimento lento e seria o substituto original do Unix?,GNU Hurd.
"No contexto de ajuda, o que representam colchetes [ ] em uma declaração de sintaxe de comando?",Itens opcionais.
"Na sintaxe de comandos, o que indicam as reticências (...) após um item?",Uma lista de comprimento arbitrário de itens daquele tipo.
A versão do kernel Linux citada no manual como exemplo é a $\text{2.6.32}$. O que o comando file diria sobre o arquivo /bin/passwd?,Que é um objeto compartilhado ELF de 64 bits para GNU/Linux $\text{2.6.32}$.
Qual comando no GNOME pode ser usado para abrir a documentação de ajuda do sistema?,O comando yelp (ou pressionando F1).
Qual empresa comprou a Red Hat por aproximadamente 34 bilhões de dólares?,IBM.
De onde vêm os codinomes das versões da distribuição Debian?,Personagens do filme Toy Story.
Qual é a lógica de nomenclatura das versões do Android?,Nomes de doces em ordem alfabética.
Qual é a lógica de nomenclatura das versões do Ubuntu?,Um adjetivo e um nome de animal seguindo a ordem alfabética.
Por que a AT&T distribuía o Unix gratuitamente para universidades nos anos 70?,Devido a um processo de monopólio que a impedia de comercializar software de computador.
Qual é o nome da fundação que hoje gerencia e sustenta o desenvolvimento do Linux?,Linux Foundation.
Qual atalho de teclado permite pular para o início da palavra anterior na linha de comando?,Ctrl+Seta para a Esquerda.
Qual atalho de teclado permite pular para o início da próxima palavra na linha de comando?,Ctrl+Seta para a Direita.
